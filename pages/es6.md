# 基本概念
>1.ECMAScript 6.0（以下简称 ES6）是 JavaScript 语言的下一代标准，已经在2015 年 6 月正式发布了。它的目标，是使得 JavaScript 语言可以用来编写复杂的大型应用程序，成为企业级开发语言。
